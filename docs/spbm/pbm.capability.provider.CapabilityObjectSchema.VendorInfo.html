Information about vendor/owner of the capability metadata schema

Properties

Name Type Description
infoPbmExtendedElementDescription

Captures name and description information about the vendor/owner of the schema.
  • The summary.label property (label) contains vendor name information in server locale.
  • The summary.summary property (summary) contains vendor description string in server locale.
  • The summary.messageCatalogKeyPrefix property (messageCatalogKeyPrefix) contains unique prefix for the vendor information within given message catalog.
