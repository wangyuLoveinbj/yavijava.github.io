Enum - PbmComplianceStatus(pbm.compliance.ComplianceResult.ComplianceStatus)


Enum Description

The PbmComplianceStatus enumeration type defines the set of status values for compliance operations. See PbmComplianceResult and PbmRollupComplianceResult.

Enum Constants

NameDescription
compliant Entity is in compliance.
nonCompliant Entity is out of compliance.
notApplicable Compliance computation is not applicable for this entity, because it does not have any storage requirements that apply to the object-based datastore on which this entity is placed.
unknown Compliance status of the entity is not known.
