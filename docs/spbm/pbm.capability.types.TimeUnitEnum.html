Enum - PbmCapabilityTimeUnitType(pbm.capability.types.TimeUnitEnum)


Enum Description

The PbmCapabilityTimeUnitType enumeration type defines the supported list of time units for profiles that specify time span capabilities and constraints. See PbmCapabilityTimeSpan.
