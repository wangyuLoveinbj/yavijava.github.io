Data Object Description

The PbmCapabilityInstance data object defines a storage capability instance. Metadata for the capability is described in PbmCapabilityMetadata.

Properties

Name Type Description
constraintPbmCapabilityConstraintInstance[]

Constraints on the properties that comprise this capability. Each entry represents a constraint on one or more of the properties that constitute this capability. A datum must meet one of the constraints to be compliant.
idPbmCapabilityMetadataUniqueId

Identifier for the capability. The identifier value corresponds to PbmCapabilityMetadata.id.
